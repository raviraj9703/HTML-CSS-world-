Styles and Semantics Tags:

  • Style Tag:

    • Tag: < style >
    Description: Defines style information for a document
    Syntax:
    < style type="name" > _ _ _ _ _ _ _ < /style >
    Example:
    < style >
    h1 {color:red;}
    p {color:blue;}
    < /style >

  • Division Tag:

    • Tag: < div >
    Description: Defines a section in a document
    Syntax:
    < div > _ _ _ _ _ _ _ < /div >
    Example:
    < div class="myDiv" >
    < h2 >This is a heading in a div element< /h2 >
    < p >This is some text in a div element.< /p >
    < /div >

  • Span Tag:

    • Tag: < span > (Similar to < div > tag)
    Description: Defines a section in a document
    Syntax:
    < span clas=" " > _ _ _ _ _ _ _ < /span >
    Example:
    < p >My mother has < span style="color:blue" >blue< /span > eyes.< /p >

  • Header Tag:

    • Tag: < header >
    Description: Defines a header for a document or section
    Syntax:
    < header > _ _ _ _ _ _ _ < /header >
    Example:
    < header >
    < h1 >A heading here< /h1 >
    < p >Posted by John Doe< /p >
    < p >Some additional information here< /p >
    < /header >

  • Footer Tag:

    • Tag: < footer >
    Description: Defines a footer for a document or section
    Syntax:
    < footer > _ _ _ _ _ _ _ < /footer >
    Example:
    < footer >
    < p >Author: Hege Refsnes< /p >
    < p >< a href="mailto:hege@example.com" >hege@example.com< /a >

    < /footer >

  • Main Tag:

    • Tag: < main >
    Description: Defines the main content for a document
    Syntax:
    < main > contents of main elements < /main >
    Example:
    < main >
    < h1 >Most Popular Browsers< /h1 >
    < p >Chrome, Firefox, and Edge are the most used browsers today.< /p >
    < article >
    < h2 >Google Chrome< /h2 >
    < p >Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today! < /p >
    < /article >
    < /main >


  • Article Tag:

    • Tag: < article >
    Description: Defines an article
    Syntax:
    < article > _ _ _ _ _ _ _ < /article >
    Example:
    < article >
    < h2 >Google Chrome< /h2 >
    < p >Google Chrome is a web browser developed by Google, released in 2008. Chrome is the world's most popular web browser today!< /p >
    < /article >


  • Aside Section Tag:

    • Tag: < aside >
    Description: Defines content aside from the page content
    Syntax:
    < aside >
    < h1 >_ _ _ _ _ _ _ < /h1 >
    < p > _ _ _ _ _ _ _ _ < /p >
    < /aside >
    Example:
    < aside >
    < h4 >Epcot Center< /h4 >
    < p >Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.< /p >
    < /aside >


  • Detail Tag:

    • Tag: < details >
    Description: Defines additional details that the user can view or hide
    Syntax:
    < details >
    _ _ _ _ _ _ _
    _ _ _ _ _ _ _
    < /details >
    Example:
    < details >
    < summary >Epcot Center< /summary >
    < p >Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.< /p >
    < /details >


  • Dialog Tag:

    • Tag: < dialog >
    Description: Defines a dialog box or window
    Syntax:
    < dialog > _ _ _ _ _ _ _ < /dialog >
    Example:
    < dialog open >This is an open dialog window< /dialog >


  • Summary Tag:

    • Tag: < summary >
    Description: Defines a visible heading for a < details > element
    Syntax:
    < summary > _ _ _ _ _ _ _ < /summary >
    Example:
    < details >
    < summary >Epcot Center< /summary >
    < p >Epcot is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.< /p >
    < /details >
