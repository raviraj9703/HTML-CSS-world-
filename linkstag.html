Links Tags:

  • Anchor Tag:

    • Tag: < a >
    Description: Defines a hyperlink
    Syntax:
    < a href= url > name < /a >
    Example:
    < a href="https://www.html and css.com" >Visit html and css .com!< /a >

  • Link Tag:

    • Tag: < link >
    Description: Defines the relationship between a document and an external resource (most used to link to style sheets)
    Syntax:
    < link rel= relationship name href= url >
    Example:
    < head >
    < link rel="stylesheet" href="styles.css" >
    < /head >

  • Navigation Tag:

    • Tag: < nav >
    Description: Defines navigation links
    Syntax:
    < nav >
    < a href= url > name < /a >
    < /nav >
    Example:
    < nav > < a href="/html/" >HTML< /a >
    < a href="/css/" >CSS< /a >
    < a href="/js/" >Publish< /a >
    < /nav >
