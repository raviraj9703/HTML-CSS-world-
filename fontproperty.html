Font Properties:

  • Font Property

    • Description:
    Sets all the font properties in one declaration
    Values:
    Font-style, font-variant, font-weight, font-size/line-height, font-family, caption, icon, menu, message-box, small-caption, status-bar, inherit

    Syntex:
    font: Font-style| font-variant| font-weight| font-size/line-height| font-family| caption| icon| menu| message-box| small-caption| status-bar| inherit;

    Example:
    p.a {
    font: 15px Arial, sans-serif;
    }
    p.b {
    font: italic small-caps bold 12px/30px Georgia, serif;
    }


  • Font-family Property

    • Description:
    Specifies the font family for text
    Values:
    Family-name, generic-family, inherit

    Syntex:
    font-family: Family-name| generic-family| inherit;

    Example:
    p.a {
    font-family: "Times New Roman", Times, serif;
    }
    p.b {
    font-family: Arial, Helvetica, sans-serif;
    }


  • Font-size Property

    • Description:
    Specifies the font size of text
    Values:
    xx-small, x-small, small, medium, large, x-large, xx-large, smaller, larger, length, %, inherit

    Syntex:
    font-size: xx-small| x-small| small| medium| large| x-large| xx-large| smaller| larger| length| %| inherit;

    Example:
    div.a {
    font-size: 15px;
    }
    div.b {
    font-size: large;
    }
    div.c {
    font-size: 150%;
    }


  • Font-style Property

    • Description:
    Specifies the font style for text
    Values:
    Normal, italic, oblique, inherit

    Syntex:
    font-style: Normal| italic| oblique| inherit;

    Example:
    p.a {
    font-style: normal;
    }
    p.b {
    font-style: italic;
    }
    p.c {
    font-style: oblique;
    }


  • Font-variant Property

    • Description:
    Specifies whether or not a text should be displayed in a small-caps font
    Values:
    Normal, small-caps, inherit

    Syntex:
    font-style: Normal| small-caps| inherit;

    Example:
    p.small {
    font-variant: small-caps;
    }


  • Font-weight Property

    • Description:
    Specifies the weight if a font
    Values:
    Normal, bold, bolder, light, 100, 200, 300, 400, 500, 600, 700, 800, 900, inherit

    Syntex:
    font-weight: Normal| bold| bolder| light| 100| 200| 300| 400| 500| 600| 700| 800| 900| inherit;

    Example:
    p.normal {
    font-weight: normal;
    }
    p.thick {
    font-weight: bold;
    }
    p.thicker {
    font-weight: 900;
    }
