Text Properties:

  • color Property

    • Description:
    Sets the color of the text
    Values:
    RGB, hex, keyword

    Syntex:
    color: RGB| hex| keyword;

    Example:
    body{
    color: blue;
    }
    h1{
    color: green;
    }


  • line-height Property

    • Description:
    Sets the distance between lines
    Values:
    Normal, number, length, %

    Syntex:
    line-height: Normal| number| length| %;

    Example:
    div.a{
    line-height: normal;
    }
    div.b{
    line-height: 1.6;
    }
    div.c{
    line-height: 80%;
    }
    div.d{
    line-height: 200%;
    }


  • letter-spacing Property

    • Description:
    Increase or decrease the space between characters
    Values:
    Normal, length

    Syntex:
    line-height: Normal| length;

    Example:
    h1{
    letter-spacing: 3px;
    }
    h2{
    letter-spacing: 2px;
    }
    h3{
    letter-spacing: -1px;
    }


  • tex-align Property

    • Description:
    Aligns the text in an element
    Values:
    Left, right, center, justify

    Syntex:
    text-align: Left| right| center| justify;

    Example:
    div.a{
    text-align: center;
    }
    div.b{
    text-align: left;
    }
    div.c{
    text-align: right;
    }
    div.d{
    text-align: justify;
    }


  • tex-decoration Property

    • Description:
    Adds decoration to text
    Values:
    None, underline, overline, linethrough

    Syntex:
    text-decoration: None| underline| overline| linethrough;

    Example:
    h1 {
    text-decoration: overline;
    }
    h2 {
    text-decoration: line-through;
    }
    h3 {
    text-decoration: underline;
    }
    h4 {
    text-decoration: underline overline;
    }


  • text-indent Property

    • Description:
    Indents the first line of text in an element
    Values:
    Length, %

    Syntex:
    text-indent:Length| %;

    Example:
    div.a {
    text-indent: 50px;
    }
    div.b {
    text-indent: -2em;
    }
    div.c {
    text-indent: 30%;
    }


  • text-transform Property

    • Description:
    Controls the letters in an element
    Values:
    None| capitalize| uppercase| lowercase

    Syntex:
    text-transform: None| capitalize| uppercase| lowercase;

    Example:
    div.a {
    text-transform: uppercase;
    }
    div.b {
    text-transform: lowercase;
    }
    div.c {
    text-transform: capitalize;
    }
