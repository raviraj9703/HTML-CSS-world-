Border Properties:

  • border Property

    • Description:
    Sets all the border properties in one declaration
    Values:
    Border-width, border-style, border-color

    Syntex:
    border: Border-width| border-style| border-color;

    Example:
    h1 {
    border: 5px solid red;
    }
    h2 {
    border: 4px dotted blue;
    }
    div {
    border: double;
    }


  • border-bottom Property

    • Description:

  • border-bottom-color Property

    Values:
    Border-color

    Example:
    div {border-bottom-color: coral;}


  • border-bottom-style Property

    • Description:
    Sets the style of the bottom border
    Values:
    none, hidden, dotted, dashed, solid, double, groove, ridge, inset, outset, initial, inherit

    Syntex:
    border-bottom-style: none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset|initial|inherit;

    Example:
    div {border-bottom-style: dotted;}


  • border-bottom-width Property

    • Description:
    Sets the width of the bottom border
    Values:
    medium, thin, thick, length, initial, inherit

    Syntex:
    border-bottom-width: medium|thin|thick|length|initial|inherit;

    Example:
    div {border-bottom-width: thin;}


  • border-color property

    • Description:
    Sets the color of the four borders
    Values:
    Color_name, hex_number, rgb_number, transparent, inherit

    Syntex:
    border-color: Color_name| hex_number| rgb_number| transparent| inherit;

    Example:
    div {border-color: coral;}
