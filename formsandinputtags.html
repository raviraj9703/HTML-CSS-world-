Forms and Input Tags

  • Form Tag:

    • Tag:< form >

    Description:
    Defines an HTML form for user input.

    Syntax:
    < form action= url method= method name >

    Example:
    < form action="/action_page.php" method="get" >
    < label for="fname" >First name:< /label > < input type="text" id="fname" name="fname">

    < label for="lname" >Last name:< /label > < input type="text" id="lname" name="lname" >

    < input type="submit" value="Submit" > < /form >


  • input Tag:

    • Tag:< input >

    Description:
    Defines an input control.

    Syntax:
    < input type= >

    Example:
    < form action="/action_page.php" method="get" >
    < label for="fname" >First name:< /label > < input type="text" id="fname" name="fname">

    < label for="lname" >Last name:< /label > < input type="text" id="lname" name="lname" >

    < input type="submit" value="Submit" > < /form >


  • Text area Tag:

    • Tag:< textarea >

    Description:
    Defines a multiline input control (text area).

    Syntax:
    < textarea id= name= rows= cols= >

    Example:
    < label for="review" >Review of HTML and CSS:< /label >
    < textarea id="review" name="review" rows="4" cols="50" >
    At w3schools.com you will learn how to make a website. They offer free tutorials in all web development technologies.
    < /textarea >


  • Clickable button Tag:

    • Tag:< button >

    Description:
    Defines a clickable click.

    Syntax:
    < button type= button/reset/submit > name < /button >

    Example:
    < button type="button" >Click Me! < /button >


  • select Tag:

    • Tag:< select >

    Description:
    Defines a drop-down list.

    Syntax:
    < select name= name form= form id >

    Example:
    < label for="cars" >Choose a car:< /label >
    < select name="cars" id="cars" >
    < option value="volvo" >Volvo< /option >
    < option value="saab" >Saab< /option >
    < option value="mercedes" >Mercedes< /option >
    < option value="audi" >Audi< /option >
    < /select >


  • Optgroup Tag:

    • Tag:< optgroup >

    Description:
    Defines a group of related options in a drop-down list.

    Syntax:
    < optgroup label= > _ _ _ _ _ _ < /optgroup >

    Example:
    < label for="cars" >Choose a car:< /label >
    < select name="cars" id="cars" >
    < optgroup label="Swedish Cars" >
    < option value="volvo">Volvo
    < option value="saab" >Saab< /option >
    < /optgroup >
    < /select >


  • Option Tag:

    • Tag:< option >

    Description:
    Defines an option in a drop-down list .

    Syntax:
    < option value= name > name < /option >

    Example:
    < label for="cars" >Choose a car:< /label >
    < select id="cars" >
    < option value="volvo" >Volvo< /option >
    < option value="saab" >Saab< /option >
    < option value="opel" >Opel< /option >
    < option value="audi" >Audi< /option >
    < /select >


  • Label Tag:

    • Tag:< label >

    Description:
    Defines a label for an < input > element.

    Syntax:
    < label for/from= name > name < /label >

    Example:
    < form action="/action_page.php" > < input type="radio" id="html" name="fav_language" value="HTML" > < label for="html" >HTML< /label >
    < input type="radio" id="css" name="fav_language" value="CSS" > < label for="css" >CSS< /label >
    < input type="radio" id="javascript" name="fav_language" value="JavaScript" > < label for="javascript" >JavaScript< /label >

    < input type="submit" value="Submit" > < /form >


  • Fieldset Tag:

    • Tag:< Fieldset >

    Description:
    Groups related elements in a form.

    Syntax:
    < fieldset >
    < label for/from= name > name < /label >
    < input type= >
    < /fieldset >

    Example:
    < html >br> < body >
    < h1 >The fieldset element< /h1 >
    < form action="/action_page.php" >
    < fieldset >
    < legend >Personalia:< /legend >
    < label for="fname" >First name:< /label >
    < input type="text" id="fname" name="fname" >
    < label for="lname" >Last name:< /label >
    < input type="text" id="lname" name="lname" >
    < label for="email" >Email:< /label >
    < input type="email" id="email" name="email" >
    < label for="birthday" >Birthday:< /label >
    < input type="date" id="birthday" name="birthday" >
    < input type="submit" value="Submit" >
    < /fieldset >
    < /form >
    < /body >
    < /html >


  • legend Tag:

    • Tag:< legend >

    Description:
    Defines a caption for a < fieldset > elements.

    Syntax:
    < legend > name < /legend >

    Example:
    < html >br> < body >
    < h1 >The fieldset element< /h1 >
    < form action="/action_page.php" >
    < fieldset >
    < legend >Personalia:< /legend >
    < label for="fname" >First name:< /label >
    < input type="text" id="fname" name="fname" >
    < label for="lname" >Last name:< /label >
    < input type="text" id="lname" name="lname" >
    < label for="email" >Email:< /label >
    < input type="email" id="email" name="email" >
    < label for="birthday" >Birthday:< /label >
    < input type="date" id="birthday" name="birthday" >
    < input type="submit" value="Submit" >
    < /fieldset >
    < /form >
    < /body >
    < /html >


  • DataList Tag:

    • Tag:< datalist >

    Description:
    Specifies a list of pre-defined options for input controls.

    Syntax:
    < datalist id= name >
    < option value= name >
    < option value= name >
    < option value= name >
    < /datalist >

    Example:
    < datalist id="browsers" >
    < option value="Edge" >
    < option value="Firefox" >
    < option value="Chrome" >
    < option value="Opera" >
    < option value="Safari" >
    < /datalist >


  • Key pair Tag:

    • Tag:< keygen >

    Description:
    Defines a key pair generator field (for forms).

    Syntax:
    < keygen name= name >

    Example:
    < form action="demo_keygen.asp" method="get" >
    Username: < input type="text" name="usr_name" >
    Encryption: < keygen name="security" >
    < input type="submit" >
    < /form >


  • Output Tag:

    • Tag:< output >

    Description:
    Defines the result of a calculation.

    Syntax:
    < output name= for/form/name > < /output >

    Example:
    < form oninput="x.value=parseInt(a.value)+parseInt(b.value)" >
    < input type="range" id="a" value="50" >
    +< input type="number" id="b" value="25" >
    =< output name="x" for="a b" >< /output >
    < /form >
