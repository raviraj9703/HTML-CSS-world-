Table Tags

  • Table Tag:

    • Tag: < table >

    Description:
    Defines a table.

    Syntax:
    < table [align =left|centre|right][border=n][cellspacing=n][cell padding=n][width=nn%][bg color= ][border color= ] >
    _ _ _ _ _ _ _
    _ _ _ _ _ _ _
    < /table >

    Example:
    < table >
    < tr >
    < th >Month< /th >
    < th >Savings< /th >
    < /tr >
    < tr >
    < td >January< /td >
    < td >$100< /td >
    < /tr >
    < /table >


  • caption Tag:

    • Tag: < caption >

    Description:
    Defines a table caption.

    Syntax:
    < caption > _ _ _ _ _ _ _< /caption >

    Example:
    < table >
    < caption >Monthly savings< /caption >
    < tr >
    < th >Month< /th >
    < th >Savings< /th >
    < /tr >
    < tr >
    < td >January< /td >
    < td >$100< /td >
    < /tr >
    < /table >


  • Header Tag:

    • Tag: < th >

    Description:
    Defines a header cell in a table.

    Syntax:
    < th > _ _ _ _ _ _ _ _< /th >

    Example:
    < table >
    < tr >
    < th >Month
    < th >Savings
    < /tr >
    < /table >


  • Row Tag:

    • Tag: < tr >

    Description:
    Defines a row in a table.

    < tr > _ _ _ _ _ _ _ _< /tr >

    Example:
    < table >
    < tr >
    < th >Month
    < th >Savings
    < /tr >
    < /table >


  • Table Date Tag:

    • Tag: < td >

    Description:
    Defines a data in cell in a table.

    Syntax:
    < td > _ _ _ _ _ _ _ _< /td >

    Example:
    < table >
    < tr >
    < td >Cell C< /td >
    < td >Cell D< /td >
    < /tr >
    < /table >


  • Table head Tag:

    • Tag: < thead >

    Description:
    Groups the header content in a table.

    Syntax:
    < thead >
    < th >_ _ _ _ _ _ _ _< /th >
    < th >_ _ _ _ _ _ _ _< /th >
    < /thead >


    Example:
    < table >
    < thead >
    < tr >
    < th >Month< /th >
    < th >Savings< /th >
    < /tr >
    < /thead >
    < /table >


  • Table Body Tag:

    • Tag: < tbody >

    Description:
    Groups the body content in a table.

    Syntax:
    < tbody > _ _ _ _ _ _ _ _ _< /tbody >


    Example:
    < tbody >
    < tr >
    < td >January< /td >
    < td >$100< /td >
    < /tr >
    < tr >
    < td >February< /td >
    < td >$80< /td >
    < /tr >
    < /tbody >

  • Column Tag:

    • Tag: < col >

    Description:
    Specifies column properties for each column within a < colgroup > element .

    Syntax:
    < col > _ _ _ _ _ _ _ _ _< /col >


    Example:
    < col span="2" style="background-color:red" >
    < col style="background-color:yellow" >


  • Column Group Tag:

    • Tag: < colgroup >

    Description:
    Specifies a group of one or more columns in a table for formatting.

    Syntax:
    < colgroup >
    < col > _ _ _ _ _ _ < /col >
    < col > _ _ _ _ _ _ < /col >
    < /colgroup >


    Example:
    < colgroup >
    < col span="2" style="background-color:red" >
    < col style="background-color:yellow" >
    < /colgroup >
