List Properties:

  • list-style Property

    • Description:
    Sets all the properties for a list in one declaration
    Values:
    List-style-type, list-style-position, list-style-image, inherit

    Syntex:
    list-style: list-style-type| list-style-position| list-style-image| inherit;

    Example:
    ul { list-style: square inside url("sqpurple.gif"); }


  • list-style-image Property

    • Description:
    Specifies an image as the list-item marker
    Values:
    URL, none, inherit

    Syntex:
    list-style-image: URL| none| inherit;

    Example:
    ul {
    list-style-image: url('sqpurple.gif');
    }


  • list-style-position Property

    • Description:
    Specifies where to place the list-item marker
    Values:
    Inside, outside, inherit

    Syntex:
    list-style-position: Inside| outside| inherit;

    Example:
    ul.a {
    list-style-position: outside;
    }
    ul.b {
    list-style-position: inside;
    }


  • list-style-type Property

    • Description:
    Specifies the type of list-item marker
    Values:
    None, disc, circle, square, decimal, decimal-leading-zero, aremenian, Georgian, lower-alpha, upper-alpha, lower-greek, lower-latin, upper-latin, lower-roman, upper-roman, inherit

    Syntex:
    list-style-type: None| disc| circle| square| decimal| decimal-leading-zero| aremenian| Georgian| lower-alpha| upper-alpha| lower-greek| lower-latin| upper-latin| lower-roman| upper-roman| inherit;

    Example:
    ul.a {list-style-type: circle;}
    ul.b {list-style-type: square;}
    ol.c {list-style-type: upper-roman;}
    ol.d {list-style-type: lower-alpha;}
