Formatting Tags:

  • Acronym Tag:

    • Tag: < acronym >
    Description: Defines an abbreviation or an acronym (not supported in HTML5. Use < abbr > instead)
    syntax:
    < acronym title=" " > name < /acronym >
    Example:
    < p >The < acronym title="World Wide Web" >WWW< /acronym > is only a component of the Internet< /p >


  • Abbreviation Tag:

    • Tag: < abbr >
    Description: Defines an abbreviation or an acronym
    syntax:
    < abbr title=" " > name < /abbr >
    Example:
    The < abbr title="World Health Organization" >WHO< /abbr > was founded in 1948.


  • Address Tag:

    • Tag: < address >
    Description:Defines contact information for the author/owner of a document/article
    syntax:
    < address > address_ _ _ _ _ _ _ _< /address >
    Example:
    < address >
    Written by < a href="mailto:webmaster@example.com" >Jon Doe< /a >.
    Visit us at:
    Example.com
    Box 564, Disneyland
    USA < /address >

  • Bold Tag:

    • Tag: < b >
    Description: Defines bold text
    syntax:
    < b > _ _ _ _ _ _ _ _ < /b >
    Example:
    < p >This is normal text - < b >and this is bold text< /b >< /p >

  • Bi-directional isolation Tag:

    • Tag: < bdi >
    Description:Isolates a part of text that might be formatted in a different form other text outside it
    syntax:
    < bdi > _ _ _ _ _ _ _ _ < /bdi >
    Example:
    < ul >
    < li >User < bdi >hrefs< /bdi >: 60 points< /li >
    < li >User < bdi >jdoe< /bdi >: 80 points< /li >
    < li >User < bdi >إيان< /bdi >: 90 points< /li >
    < /ul >


  • Big Tag:

    • Tag: < big >
    Description: Defines big text (not supported in HTML5. Use CSS instead)
    syntax:
    < big > _ _ _ _ _ _ _ _ < /big >
    Example:
    Big Text is < big > bigger< /big > than regular text.


  • Block quotation Tag:

    • Tag: < blockquote >
    Description: Defines a section that is quoted from another source
    syntax:
    < blockquote > _ _ _ _ _ _ _ _ < /blockquote >
    Example:
    < blockquote cite="http://www.worldwildlife.org/who/index.html" >
    For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
    < /blockquote >

  • Center alignment Tag:

    • Tag: < center >
    Description: Defines centered text (not supported in HTML5. Use CSS instead)
    syntax:
    < center > _ _ _ _ _ _ _ _ < /center >
    Example:
    < center >
    < p > Your centered text goes here < /p >
    < /center >
  • Cite Tag:

    • Tag: < cite>
    Description: Defines the title of work
    syntax:
    < cite > _ _ _ _ _ _ _ _ < /cite >
    Example:
    < p >< cite >The Scream< /cite > by Edward Munch. Painted in 1893.< /p >

  • Code Tag:

    • Tag: < code>
    Description: Defines a piece of computer code
    syntax:
    < code > _ _ _ _ _ _ _ _ < /code >
    Example:
    < p >The HTML < code >button< /code > tag defines a clickable button.< /p >
    < p >The CSS < code >background-color< /code > property defines the background color of an element.< /p >


  • Delete Tag:

    • Tag: < del >
    Description: Defines text that has been deleted from a document
    syntax:
    < del > _ _ _ _ _ _ _ _ < /del >
    Example:
    < p >My favorite color is < del >blue< /del > < ins >red< /ins >!< /p >


  • Definition element Tag:

    • Tag: < dfn >
    Description: Represents the defining instance of a term
    syntax:
    < dfn > _ _ _ _ _ _ _ _ < /dfn >
    Example:
    < p >< dfn >HTML< /dfn > is the standard markup language for creating web pages.< /p >

  • Emphasized text Tag:

    • Tag: < em >
    Description: Defines emphasized text
    syntax:
    < em > _ _ _ _ _ _ _ _ < /em >
    Example:
    < p >You < em >have< /em > to hurry up!< /p >
    < p >We < em >cannot< /em > live like this.< /p >
  • Font Tag:

    • Tag: < font >
    Description: Defines font, color, and size for text (not supported in HTML5. Use CSS instead)
    syntax:
    < font size= " " color=" " face=" " > _ _ _ _ _ _ _ _ < /font >
    Example:
    < p style="font-family: verdana" >This is a paragraph.< /p >
    < p style="font-family :”Courier New” >This is another paragraph.< /p >


  • Italic Tag:

    • Tag: < i >
    Description: Defines a part of text in an alternate voice or mood
    syntax:
    < i > _ _ _ _ _ _ _ _ < /i >
    Example:
    < p >< i >Lorem ipsum< /i > is the most popular filler text in history.< /p >
    < p >The < i >RMS Titanic< /i >, a luxury steamship, sank on April 15, 1912 after striking an iceberg.< /p >


  • Insert Tag:

    • Tag: < ins >

    syntax:
    < ins > _ _ _ _ _ _ _ _ < /ins >
    Example:
    < p >My favourite color is < del >blue< /del > < ins >red< /ins >!< /p >


  • Keyboard Input Tag:

  • Meter Tag:

    • Tag: < meter >
    Description: Defines a scalar measurement within a known range ( a gauge)
    syntax:
    < meter form=" " high=" " low=" " max=" " min=" " optimum=" " value=" " > _ _ _ _ _ _ _ _ < /meter >
    Example:
    < label for="disk_c" >Disk usage C:< /label > < meter id="disk_c" value="2" min="0" max="10" >2 out of 10< /meter >

    • Tag: < pre >
    Description: Defines preformatted text

    < pre > _ _ _ _ _ _ _ _ < /pre >
    Example:

    Text in a pre element
    is displayed in a fixed-width

    both spaces and

    < /pre >

  • Progress Tag:

    • Tag: < progress >
    Description: Represents the progress of a task

    • Tag: < q >
    Description: Defines a short quotation

  • Ruby Tag:

    • Tag: < rp >
    Description: Defines what to show in browsers that do not support ruby annotations
    syntax:
    < rt >
    < rp >[< /rp> _ _ _ _ _ _ < rp >]< /rp >
    < /rt >
    Example:
    < body >
    < h1 > HyperText Markup Language< /h1 >
    < h2 >< ruby > Tag< /h2 > < ruby >HTML< rt > HyperText Markup Language< /rt >< /ruby >
    < /body >

  • Ruby Tag:

    • Tag: < ruby >
    Description: Defines a ruby annotation (for east Asian typography)
    syntax:
    < ruby < rt >/< rp > > _ _ _ _ _ _ _ < /ruby >
    Example:
    < ruby >
    Tag< rt > HTML < /rt >
    < /ruby >
