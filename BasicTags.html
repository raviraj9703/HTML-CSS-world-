Basic Html Tags


  • Html Tag:

    • Tag: < Html >

    Description:
    Defines an HTML document.

    syntax :
    < html > number of sub tags < /html >

    Example:
    < html >
    < head >< meta >< title >Title of your webpage< /title >< /head >
    < body >Webpage contents< /body >
    < /html >
  • Title Tag:

    • Tag: < title >

    Descripion:
    Defines a title for the document.

    syntax:
    < title >_ _ _ _ _ _ < /title >

    Example:
    < TITLE >Title of your webpage< /TITLE >


  • Head Tag:

    • Tag: < Head >

    Description:
    Defines a head information.

    Syntax:
    < head > _ _ _ _ _ _ < /head >

    Example:
    < HEAD >Contains elements describing the document< /HEAD >


  • Body Tag:

    • Tag:< body >

    Description:
    Defines the document's body.

    Syntax:
    < body > _ _ _ _ _ _ < /body >

    Example:
    < BODY >The content of your page< /BODY >


  • Headings Tag:
    • Tag: < h1 to h6 >

    Description:
    Defines HTML headings.

    Syntax:
    < h1[align =left|centre|right] > -----< /h1 >
    < h2[align =left|centre|right] > -----< /h2 >
    < h3[align =left|centre|right] > -----< /h3 >
    < h4[align =left|centre|right] > -----< /h4 >
    < h5[align =left|centre|right] > -----< /h5 >
    < h6[align =left|centre|right] > -----< /h6 >

    Example:
    < H1 >Heading 1 Example< /H1 >
    < H2 >Heading 2 Example< /H2 >
    < H3 >Heading 3 Example< /H3 >
    < H4 >Heading 4 Example< /H4 >
    < H5 >Heading 5 Example< /H5 >
    < H6 >Heading 6 Example< /H6 >


  • Paragraph Tag:

    • Tag : < p >

    Descripation:
    Defines a paragraph.

    Syntax:
    < p >_ _ _ _ _ _ _ _ _ _ _ _ _< /P >

    Example:
    < p >This is some text in a paragraph< /p >


  • Break Tag

    • Tag: < br >

    Descripation:
    Inserts a single line break.

    Syntax:
    < br > (ending tag(< /br >) is optional)

    Example:
    < p >To force< br >line breaks< br >in a text, < br >use the br< br > element< /p >


  • Horizontal Tag:

    • Tag: < hr >

    Descripation:
    Defines a thematic change in the content.

    Syntax:
    < hr[align =left|centre|right][size=n][no shade][width=nn%] >

    Example :
    < hr width=25% color=#6699FF size=6 >
